### Cosmos SDK V0.53 Notice
Cosmos SDK v0.53 includes various bug fixes and improvements

#### What Changed Upstream
Bug Fixes:
- Fixed a bug in `distribution` module that could lead to incorrect reward calculations and chain halts.
- Fixed a bug in `FinalizeBlock` that could cause a chain halt.
- Fixed a bug where `null` pagination parameters in queries triggered a `500` error code and crashed for nodes processing the request.
- Fixed a bug where `DenomMetadata` RPC requests incorrectly parsed IBC denoms containing slashes (e.g., `ibc/123ABC`), resulting in a 404 error code.
- Fixed a bug in `staking` module where invalid input or inconsistent state conditions could lead to a chain halt.
